# AI Prompt Engineering Samples

## About
This repository contains examples of prompt writing, prompt refinement, and response verification to improve the quality, accuracy, and clarity of AI-generated responses.

## Example 1: Resume Prompt

### Initial Prompt
Create a resume summary for a graduate.

### Refined Prompt
Create a concise and professional resume summary for a Clinical Nutrition graduate applying to analyst and operations roles. Highlight communication skills, MS Office, and customer handling abilities.

### Outcome
The response became more professional, role-specific, and aligned with job requirements.

## Example 2: Interview Answer Prompt

### Initial Prompt
Answer why I want to work at a company.

### Refined Prompt
Write a concise and professional interview response explaining interest in a company, focusing on growth, contribution, and learning.

### Outcome
The response became clearer, more structured, and professional.

## Example 3: Response Verification

### Task
Review an AI-generated response for grammar, clarity, tone, and relevance.

### Verification Criteria
- Accuracy
- Professional tone
- Clear communication
- Relevance

### Outcome
Improved readability, quality, and correctness of responses.
